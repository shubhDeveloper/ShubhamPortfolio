About Me

Hi there! I'm all about making sense of data with tools like Power BI and Tableau. But my real knack is using the magic of artificial intelligence to make analytics even stronger. By teaming up AI with data, I help find deeper insights for smarter decisions, ensuring data works like a well-oiled machine.

Oracle development is my passion. Whether it's building strong databases using Oracle, MS SQL Server, or creating user-friendly applications with Oracle APEX, I excel. I'm skilled in Oracle APEX, which lets me create dynamic apps that work seamlessly with Oracle databases.

Wrangling data is my superpower. Whether it's data warehousing, data modeling, or ETL wizardry, I've got it covered. Curiosity fuels my passion, driving me to keep learning and bring data to life in meaningful ways. Join me on this journey where data meets innovation!
